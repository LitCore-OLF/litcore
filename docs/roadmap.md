# LitCore Practical Roadmap

This roadmap prioritizes practical adoption over format evangelism.

## Phase 1 — Reliability & Spec (now)

- Publish and maintain a versioned OLF specification.
- Expand conformance tests around parser and AST behavior.
- Preserve stable block-level anchors for citation/diff workflows.

## Phase 2 — Interoperability

- Implement robust Markdown import/export.
- Add EPUB/TEI/Docx adapters (initially as best-effort bridges).
- Define lossiness expectations per adapter.

## Phase 3 — Query & Discovery

- Upgrade search beyond substring matching (tokenized indexing).
- Add explicit indexing strategy for large corpora.
- Provide query examples for scholarly and AI data workflows.

## Phase 4 — Governance & Trust

- Publish compatibility policy for v1.x.
- Add RFC process for syntax/runtime changes.
- Maintain transparent release notes and migration docs.

## Phase 5 — Adoption Proof

- Ship multiple production-grade reference books.
- Demonstrate multilingual + long-form + collaborative pipelines.
- Publish reproducible benchmarks and methodology.
