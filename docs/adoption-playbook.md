# LitCore Adoption Playbook

## Positioning

Lead with one concrete message:

**“Git-native structured literature for AI-ready corpora and scholarly citation.”**

## Near-Term Distribution Steps

1. Publish 2–3 end-to-end reference repositories using public-domain texts.
2. Publish direct comparisons: LitCore vs Markdown/EPUB/TEI (tradeoffs, not hype).
3. Provide integration examples for Notion, static site pipelines, and Python consumers.
4. Run a pilot with 1–2 focused communities (digital humanities, translation teams, canonical text projects).
5. Document measurable outcomes (onboarding speed, query quality, citation stability, diff quality).

## Community Flywheel

- Keep a public roadmap.
- Curate “good first issues”.
- Add contributor templates for new books and bridges.
- Share monthly implementation updates with examples, not claims.
